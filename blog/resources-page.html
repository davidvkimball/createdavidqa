Resources Page

Thinking through how the resources page should work on this site.

I'm envisioning a "Resources" option along the top nav, with a dropdown menu that include items for:

  • Neocities
  • Spacehey
  • Firefox
  • Free Tools
  • Windows

I'm worried, however, that this will be duplicate or redundant with the links section.

Since create.dvdv.kim is a hub of sorts, I will be spending lots of time linking to other places... and maybe that's OK. No sense in reinventing the wheel when an external source already does it best.

This continues to be a work-in-progress, and as always, please continue sharing your feedback and suggestions so I can make the site better!
